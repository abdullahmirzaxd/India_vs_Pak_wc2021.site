# T20 World Cup 2021 - Pakistan vs India Match Web App

This is a Streamlit web application that displays detailed statistics and analysis of the iconic T20 World Cup 2021 match between Pakistan and India.

## Features
- Display scorecards for both teams (batting and bowling).
- Visualize player performances using bar charts.
- Track the run rate progression over the course of the match.

## Requirements
To run this app, you'll need the following Python libraries:
- streamlit
- pandas
- plotly

Install these dependencies using:
```bash
pip install -r requirements.txt
